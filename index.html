Output variable: SleepHours

⏰ SHORTCUT 2 — MORNING AUTOMATION

Runs Shortcut 1 automatically every morning. No tapping required — wake up, see your readiness score already waiting.

2

🏋️ SHORTCUT 3 — POST-WORKOUT AUTO-LOGGER

After you finish a session in this app, your workout data is automatically copied to clipboard. This Shortcut reads it and logs to Apple Health with zero prompts — no duration/calorie questions.

1
Open Shortcuts → tap + → name it:
5/3/1 Auto Logger
2
Add action: Get Clipboard
→ Rename output: WorkoutJSON
3
Add action: Get Dictionary from Input
Input: WorkoutJSON
→ Rename output: WorkoutData
4
Add action: Get Value for Key in Dictionary
Dictionary: WorkoutData
Key: duration → output: Duration

Repeat for key: calories → output: Calories
5
Add action: Log Workout (Health)
Activity Type: Traditional Strength Training
Duration: Duration variable (minutes)
Active Energy: Calories variable
Date: Current Date
6
Add action: Show Notification
Title: Workout Logged ✓
Body: Duration min · Calories kcal → Apple Health
7
Save. In this app, finish a session → the sync modal now has "📋 Auto-Log to Health" which copies workout JSON to clipboard and launches this Shortcut automatically.
